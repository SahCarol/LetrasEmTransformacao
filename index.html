📜 Regras do jogo

Objetivo: Criar textos literários a partir de problema social, letra (com palavra associada) e gênero.

Jogadores: 2 a 6 (ou solo/cooperativo).

  • 1. Sortear – clique em "Sortear combinação". Aparecerá: problema social + letra (com 3 palavras) + gênero (terror, comédia, romance, aventura, suspense, drama, ficção científica, fantasia).
  • 2. Escolher uma palavra – das 3 sugeridas para a letra, o jogador escolhe apenas uma (não precisa usar todas).
  • 3. Criar (1:30) – use o timer opcional. Texto curto (poema, miniconto, cena).
  • 4. Debate – sem tempo limite, o grupo discute a relação com o problema social.
  • 5. Pontuar – outros jogadores dão +1 (pertinente), +2 (criativo) ou +3 (impactante).
  • 6. Próximo – novo sorteio, nova rodada.

Fim do jogo: Após 4-6 rodadas, soma-se pontos. Quem tiver mais pontos vence. Modo cooperativo: criar um "livro de ativismo literário".
